# Rails 5 Template

This is a template for building new rails 5 applications with my defaults. It uses haml, bourbon, neat, font-awesome, rspec (instead of MiniTest) and generates a basic site structure.

## Gems

- haml
- bourbon
- neat
- font-aweseom-rails
- rspec
- factory_girl
- capistrano
- twitter-bootstrap v4?
